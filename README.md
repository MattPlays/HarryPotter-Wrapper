# HarryPotter.js
harrypotter.js is a Javascript wrapper for the [HP-API](https://hp-api.herokuapp.com)
 
# Installation
```bash
npm install harrypotter.js
```

# Usage
**JavaScript**
```javascript
const {hp} = require("harrypotter.js");
```
**TypeScript**
```typescript
import {hp} from "harrypotter.js";
```
